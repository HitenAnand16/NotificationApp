# NotificationApp

NotificationApp is a React Native application that allows users to send messages and receive push notifications. This project is built using Expo and React Navigation.

## Features

- Send a message through a simple UI
- Receive push notifications with the sent message
- Smooth animations and modern UI design
- Works on both Android and iOS devices

## Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/HitenAnand16/NotificationApp.git
   cd NotificationApp

2. Install dependencies:
    npm install

3. Run the application:
    npx expo start.
